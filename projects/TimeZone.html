TimeZone

  • Tech Stack: Python, discord.py, Google API, Time and Weather API, Latitude and Longitude API, MySQL, Heroku.
  • Bot Profile on Discord: Bot profile link on Discord
  • Github URL: Project Link

A python based event co-ordination bot that also doubles as a to-do list. This bot has an international range as it is on the Discord app which is widely used.

Used MYSQL database to store the latitute and longitude of the user which are later fed to the Time and Date API to be used later for better co-ordination.

It also has a to-do list feature which can be used for better event planning as well as day to day planning.
