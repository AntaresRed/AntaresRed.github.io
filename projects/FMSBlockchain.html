File Management System using Blockchain Technology

  • Tech Stack: Python, MySQL, pycryptodome, json5, jsonpickle, Flask, Flask-Classful, p2pnetwork and requests.
  • Link to paper: Research Paper

Constructed a template for a File Management system using Blockchain’s Proof of Stake Consensus algorithm to prevent data leaks.

The system mainly consists of 3 modules: ”File Security Module” which performs the cryptographic tasks, a ”World State Module” which restricts access to sensitive files, and a ”Blockchain Module” which checks and clears all the file transactions that occur so that files are never misplaced into uncleared systems.ss

The File Security module consists of file encryption and decryption sub modules which secure the file before transmission.

The World State module runs a timed full system scan of the user system to detect uncleared and sensitive files to maintain the file hierarchy.

The Blockchain module acts as the clearance checker and token distributor in the system and is the cornerstone of my project.

To understand the project in greater detail, please read the research paper.
